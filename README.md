# FinaTrack
A Digital Automation Tool for Personal Expense Management

## Overview
FinaTrack is a personal automation project developed for a university Digital Business exam.
It automates the aggregation of income and expenses and calculates remaining monthly budgets.

## Business Value
- Automates repetitive financial calculations
- Reduces manual bookkeeping errors
- Supports data-driven personal decision-making

## Features
- Expense categorization
- Automated calculations
- Fast and reliable execution
- Linux-compatible automation tool

## Build & Run
```bash
cargo build --release
./target/release/autotrack
```

## Website
A simple static website is included to advertise and explain the tool.
